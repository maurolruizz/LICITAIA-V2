# EQUIVALENCIA NOMINAL OFICIAL — ESTRUTURA REAL DO PROJETO

## 1. Objetivo

Regularizar, sem ambiguidade, a divergência nominal identificada entre:

- `ESTRUTURA REAL DO PROJETO — LICITAIA V2.md`
- `ESTRUTURA REAL DO PROJETO — LICITAIA V2.md.txt`

## 2. Solução oficial adotada

Fica estabelecida, para todos os efeitos de governança do projeto, a seguinte equivalência normativa:

- Nome lógico-normativo: `ESTRUTURA REAL DO PROJETO — LICITAIA V2.md`
- Nome físico vigente no repositório: `01-planejamento/ESTRUTURA REAL DO PROJETO — LICITAIA V2.md.txt`

Documento oficial válido e vigente: **arquivo físico `.md.txt` acima**.

## 3. Regra de uso obrigatório

Até eventual normalização física futura, toda referência normativa à estrutura real deve considerar:

- o título lógico `...V2.md` como denominação documental; e
- o caminho físico `.md.txt` como fonte oficial de leitura e conferência.

Não há documento concorrente com autoridade equivalente fora deste caminho físico.

## 4. Efeito de conformidade

Com esta equivalência oficial registrada em governança, a divergência nominal é considerada resolvida para fins de:

- leitura obrigatória;
- auditoria;
- checkpoints normativos;
- critério 10/10 sem ambiguidade documental residual.
